GM命令大全

说明 : 命令参数有顺序需求, 分隔符是英文空格. [ ]括起来的参数 表示是可选参数

正式命令列表如下:

 

l  #account user N [passs]  创建N个账号,创建的账号如 user,user1,user2…userN-1   pass 默认是123

 

l  #getcard id [N]   获取N张指定id的卡牌. 如果已拥有此卡牌,则增加数量. 当N<0时表示扣除N张卡.  N 默认是1.  当id==0时,表示清空所有卡牌,并重新初始化卡牌

 

l  #battle user  无需匹配,直接与user玩家进入战场 (user必须在线)

 

l  #grade N  直接设置等级，N必须大于0， 当N小于当前等级，表示降级

 

l  #exp N 增加经验值，当N<0 表示扣除经验

 

l  #sp N  增加体力值，当N<0 表示减少体力

 

l  #getunit starId [grade]  获取单位，starId为s_star表id，grade为单位等级，等级可以不填默认为1级

 

l  #gold num  获得军费，可以为负数
